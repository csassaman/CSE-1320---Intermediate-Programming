## For Loop

```c++
for(INIT; CONDITION; INCREMENT) {
    // Do something
}
```

## While Loop

```c++ 
while (CONDITION) {
    // Do something
}
```

- enterence control

## Do-while loop

```c++
do {
    // Do something
} while (CONDITION);
```

## Additional Control

- `break`
  - stops processing and returns to the next higher level of scope
- `continue`
  - stops processing for the iteration and returns to the beginning of the scope
- `return`
  - exits the function
- `exit()`
  - can be bad, a cop-out
